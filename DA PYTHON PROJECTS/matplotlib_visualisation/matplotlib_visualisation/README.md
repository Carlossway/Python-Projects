# Project: Data Visualisation and Environment Setup

## Description
This project contains two Jupyter notebooks designed to provide hands-on learning for data analysis and visualisation using Python.

### Notebooks Included:
1. **Matplotlib Visualisation with Real-World Scenarios**:
   - Learn how to use Matplotlib to create effective and impactful visualisations based on real-world data.
2. **Setting Up a Virtual Environment and Using Pandas**:
   - A step-by-step guide to setting up a virtual environment, installing Pandas, and exploring its functionalities.

## Requirements
- Python 3.8 or above
- Virtualenv or venv
- Jupyter Notebook

## Getting Started
1. Clone this repository to your local machine.
2. Follow the instructions in `instructions.md` to set up your environment and run the notebooks.

## Learning Objectives
- Understand the fundamentals of creating data visualisations using Matplotlib.
- Gain practical knowledge in setting up and managing Python virtual environments.
- Learn to manipulate and analyse data with Pandas.
