
# Data Manipulation Project

## Overview
This project comprises a series of Jupyter notebooks designed to demonstrate advanced data manipulation techniques using Pandas in Python. The notebooks encompass a range of topics from setting up environments to executing complex data joins and reshaping data structures.

## Notebooks Description

1. **Virtual_Environment_and_Pandas_Setup.ipynb**
   - **Purpose:** Guides you through setting up a virtual environment and installing the Pandas library.
   - **Instructions:** Follow the steps in the notebook to create a virtual environment and install all necessary Python packages.

2. **Data_Joining_and_Indexing.ipynb**
   - **Purpose:** Demonstrates how to perform SQL-like joins and indexing in Pandas.
   - **Instructions:** Execute the notebook cells to see examples of data joining and the utilisation of indexes to optimise searches.

3. **Data_Sorting_and_Reshaping.ipynb**
   - **Purpose:** Focuses on sorting data and reshaping it using various Pandas functions.
   - **Instructions:** Run through the notebook to learn about sorting techniques and how to reshape data using methods such as `pivot()` and `pivot_table()`.

4. **Extra_Series.ipynb**
   - **Purpose:** Provides additional examples and exercises related to Pandas Series objects.
   - **Instructions:** Explore the notebook for a deeper understanding of the Series data structure in Pandas.

## Getting Started
To get started with this project, clone this repository and ensure you have Visual Studio Code installed with the Python extension. Open each notebook in Visual Studio Code, and select the appropriate kernel (ensure your virtual environment is active).

## Prerequisites
- Python 3.8+
- Pandas library
- Jupyter extension for Visual Studio Code

## Installation
1. Clone the repository to your local machine.
2. Navigate to the repository directory.
3. Create a virtual environment:
   ```
   python -m venv venv
   ```
4. Activate the virtual environment:
   - **Windows:**
     ```
     .\venv\Scripts\activate
     ```
   - **Mac/Linux:**
     ```
     source venv/bin/activate
     ```
5. Install the required packages:
   ```
   pip install -r requirements.txt
   ```

## Usage
Open Visual Studio Code, navigate to the project folder, and open the Jupyter notebooks to start experimenting with the examples provided.
