
# Instructions for Data Manipulation Project

## Getting Started
Before you begin, ensure that you have cloned the repository and have Visual Studio Code installed on your machine. You will need to use Visual Studio Code to open and run the Jupyter notebooks contained within the project.

## Virtual Environment Setup
1. **Create and Activate a Virtual Environment:**
   - Open a terminal in Visual Studio Code.
   - Run the following command to create a virtual environment:
     ```
     python -m venv venv
     ```
   - Activate the virtual environment:
     - **Windows:**
       ```
       .\venv\Scripts\activate
       ```
     - **Mac/Linux:**
       ```
       source venv/bin/activate
       ```

## Installing Dependencies
2. **Install Required Python Packages:**
   - Ensure your virtual environment is activated.
   - Install the packages necessary for the project by running:
     ```
     pip install -r requirements.txt
     ```

## Notebook Descriptions and Usage
3. **Using the Notebooks:**
   - Each notebook in the project is designed for a specific data manipulation task using Pandas.
   - Open each notebook in Visual Studio Code by navigating to the project directory and selecting the desired notebook.
   - Ensure that the Python interpreter is set to the virtual environment you created earlier.
   - Execute the cells in each notebook to see the code in action and learn about various data manipulation techniques.

### Notebooks Included:
- **Virtual_Environment_and_Pandas_Setup.ipynb:** Walks through the initial setup process including environment creation and package installations.
- **Data_Joining_and_Indexing.ipynb:** Demonstrates advanced techniques for joining and indexing data.
- **Data_Sorting_and_Reshaping.ipynb:** Teaches sorting and reshaping of data using different Pandas functionalities.
- **Extra_Series.ipynb:** Provides further exploration into the use of Pandas Series.

## Troubleshooting Common Issues
4. **Problem Solving:**
   - If you encounter issues with notebook execution, ensure that your virtual environment is active and that all dependencies are installed.
   - Check for error messages in the terminal and consult the Pandas documentation for specific function usage and troubleshooting.

## Further Learning
5. **Expand Your Skills:**
   - Once you are comfortable with the basics covered in these notebooks, try modifying the code to perform different data manipulations.
   - Experiment with additional Pandas functions and methods to deepen your understanding of data analysis.
